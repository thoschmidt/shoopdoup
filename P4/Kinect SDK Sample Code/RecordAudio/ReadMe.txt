=============================
OVERVIEW  
.............................
This module provides sample code used to demonstrate the use of the KinectAudioSource
for audio capture and beam tracking.
Audio captured is written to an out.wav WAV format file.

=============================
SAMPLE LANGUAGE IMPLEMENTATIONS     
.............................
This sample is available in C#


=============================
FILES   
.............................
- Program.cs: Defines RecordAudio.Program class and associated executable Main function

=============================
BUILDING THE SAMPLE   
.............................

To build the sample using Visual Studio:
-----------------------------------------------------------
1. In Windows Explorer, navigate to the RecordAudio\CS directory.
2. Double-click the icon for the .sln (solution) file to open the file in Visual Studio.
3. In the Build menu, select Build Solution. The application will be built in the default \Debug or \Release directory.


=============================
RUNNING THE SAMPLE   
.............................

To run the sample:
------------------
1. Navigate to the directory that contains the new executable, using the command prompt or Windows Explorer.
2. Type RecordAudio at the command line, or double-click the icon for RecordAudio.exe to launch it from Windows Explorer.
